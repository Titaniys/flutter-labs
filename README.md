\# Географический справочник



Приложение-справочник для быстрого поиска информации о странах мира. Пользователь может ввести название страны, получить о ней основные данные (столица, население, валюта и т.д.), а также просмотреть историю своих поисковых запросов.



\## Основные функции



\*   \*\*Поиск информации о стране:\*\* Ввод названия страны для получения справочных данных.

\*   \*\*Отображение данных:\*\* Вывод следующей информации:

&nbsp;   \*   Название страны

&nbsp;   \*   Столица

&nbsp;   \*   Флаг

&nbsp;   \*   Население

&nbsp;   \*   Языки

&nbsp;   \*   Валюта

\*   \*\*История поиска:\*\* Автоматическое сохранение названий запрошенных стран в список истории.

\*   \*\*Управление историей:\*\* Возможность удаления отдельных записей из истории поиска (иконка корзины).



\## Структура экранов



Приложение состоит из двух основных экранов:



1\.  \*\*Главный экран (Поиск):\*\*

&nbsp;   \*   Содержит заголовок "Географический справочник".

&nbsp;   \*   Поле ввода с плейсхолдером "Введите страну для поиска".

&nbsp;   \*   Блок для отображения результатов поиска (лейблы полей: Название страны, Столица, Флаг, Население, Языки, Валюта).

&nbsp;   

2\.  \*\*Экран истории:\*\*

&nbsp;   \*   Содержит заголовок "Географический справочник".

&nbsp;   \*   Список ранее введенных стран (например, "Страна 1", "Страна 2" и т.д.).

&nbsp;   \*   Кнопка удаления (иконка мусорной корзины) напротив каждой записи в списке.



\## Навигация



Навигация между экранами осуществляется с помощью нижней панели с двумя кнопками:



\*   \*\*Главная:\*\* Переход на экран поиска и ввода данных. При нахождении на этом экране кнопка подсвечивается (серый фон).

\*   \*\*История:\*\* Переход на экран со списком последних запросов. При нахождении на этом экране кнопка подсвечивается (серый фон).
